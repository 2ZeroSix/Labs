	баланс - модуль разницы высот поддеревьев <= 1

	Ре-балансировка при вставке состоит из вращений

		a, b, c... - листы
		A, B, C... - сбалансированные деревья

		малое левое вращение:
			До балансировки:
					a
			L				b
						C		R

			h(b) - h(L) = 2
			h(C) <= h(R)

			После балансировки:
						b
				a			R
			L		C

		Большое левое вращение:
			До балансировки:
				a
			L					b
						c				R
					M		N

			h(b) - h(L) = 2
			h(c) > h(R)
			
			После балансировки:
						c
				a				b
			L		M		N		R

		малое правое вращение:
			до балансировки:
						a
				b				R
			L		C

			h(b) - h(R) = 2
			h(C) <= h(L)

			после балансировки:
				b
			L			a
					C		R

		Большое правое вращение:
			до балансировки:
								a
				b					R
			L			c
					M		N


			h(b) - h(R) = 2
			h(c) > h(L)

			после балансировки:
						c
				b				a
			L		M		N		R

			1 поиск куда вставить
			2 вставка
			3 рекурсивный подъём по пути поиска и балансировка
